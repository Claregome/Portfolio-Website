# Personal Portfolio Website

## Overview

This project is a personal portfolio website developed to showcase my skills, projects, and background as a Computer Science student. It serves as a central platform where visitors can learn more about me, explore my work, and get in touch.

The website is designed with a clean, modern layout and is fully responsive, ensuring a smooth experience across desktop and mobile devices.

## Live Demo
You can view the live website here: https://claregome.netlify.app/

## Features
👤 About Me section highlighting my background and interests
💻 Projects section showcasing my work and GitHub repositories
🛠️ Skills section displaying my technical expertise
📱 Fully responsive design for all screen sizes
📬 Contact section for easy communication
